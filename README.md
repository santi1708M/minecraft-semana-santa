## 🛡️ Proteger Tu Territorio (Claims)

Puedes proteger tu territorio para que nadie pueda romper o construir en tu área.

### Usando la Pala Dorada (Golden Shovel)

1. **Obtené una pala dorada**: Ya viene en tu kit starter, o usá `/kit claim`
2. **Click derecho** en una esquina de tu terreno
3. **Click derecho** en la esquina opuesta
4. ¡Listo! Tu claim está protegido

### Comandos de Claim

| Comando | Descripción |
|---------|-------------|
| `/claim` | Crear un claim automático en tu posición |
| `/claimlist` | Ver tus claims |
| `/trust <jugador>` | Dar acceso a otro jugador en tu claim |
| `/untrust <jugador>` | Quitar acceso a un jugador |
| `/abandonclaim` | Eliminar un claim (pierdes los bloques) |
| `/claimslist` | Ver cuántos bloques de claim tienes |

### Explicación de permisos en un claim

| Permiso | Descripción |
|---------|-------------|
| **Access** | Puede abrir cofres pero no romper bloques |
| **Build** | Puede construir y romper bloques |
| **Container** | Puede usar cofres, hornos, etc. |
| **Owner** | Dueño completo del claim |

### Ejemplo práctico
```
/trust Pepito    -> Pepito puede construir en tu terreno
/trust Pepito access   -> Pepito solo puede abrir cofres
```

## ⚔️ PvP (Combate)

El PvP está habilitado, pero **cada jugador decide** si quiere o no recibir daño.

### Comandos de PvP

| Comando | Descripción |
|---------|-------------|
| `/pvp` | Ver tu estado de PvP |
| `/pvp on` | Activar PvP (podes recibir daño) |
| `/pvp off` | Desactivar PvP (no podes recibir daño) |

**Por defecto**: El PvP está desactivado. ¡Vos elegís!

## ❓ Preguntas Frecuentes

### ¿Cómo protejo mi casa?
Usá la pala dorada (golden shovel) de tu kit starter. Click en dos esquinas y listo.

### ¿Qué pasa si muero?
Podés usar `/back` para volver a donde moriste (si el servidor lo permite).

### ¿Puedo tener varias casas?
Sí! Usá `/sethome` tantas veces como quieras. Para ir a una específica: `/home nombre`.

### ¿Cómo puedo ganar dinero?
- Vendiendo items en la tienda (`/sell`)
- Minando minerales
- Vendiendo cosas en el `/shop`

### ¿El PvP está activo?
Sí, pero cada jugador puede elegir. Usá `/pvp off` si no querés que te maten.
